# Hotel Booking Revenue Analysis

Pricing analysis and commercial performance tracking  
on a hotel booking dataset of 119,000 reservations.

## Objectives

- Analyze seasonality and booking trends
- Measure price sensitivity by customer segment
- Calculate key indicators: ADR, RevPAR, occupancy rate
- Build a performance monitoring dashboard

## Tools

- Python (pandas, matplotlib, seaborn)
- Excel (pivot tables)
- Power BI (interactive dashboard)

## Data Source

[Hotel Booking Demand — Kaggle](https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand)
